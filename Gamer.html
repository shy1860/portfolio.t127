Gamer Lobby MEAN Application

This Gamer Lobby application is developed using MEAN Stack for players to join other online available players to play games. This application features a list of available games that the players can join if they are interested. There is an admin interface for creating and monitoring players and games.

Walk-through of the APP

Some features that were introduced in this APP:

  • Available gamers list which allows other user to view and join
  • Administrator page will allow management of the players
  • Allows user to register accounts and join other games
