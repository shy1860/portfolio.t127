MEVN Chat App

This MEVN Chat App application is deployed with heroku and also uses axios. The main page shows the list of rooms and option to join. If the user clicks on join, it asks them for their nickname and finally it takes the user to the chat room with other users..

Walk-through of the APP

Some features that were introduced in this APP:

  • Room list on main page
  • User can add room,join room,make a nick name which will appear to the other users in the same room
  • It allows user send and receive message simultaneously
