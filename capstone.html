Stu-share Android Campus information exchange APP

This is Stushare Android App used for GBC student with campus information exchange and this is easy-to-use. This App allows students to share their student life events information within a safe on campus studnet environment.

Walk-through of main screen

Some features that were introduced in this APP:

  • Swipe left/right to navigate between activities
  • Navigation bottom bar
  • Searching by the keywords in title and deescription of events
  • Sorting by various criteria
  • Navigation draweron the top left corner
Event List

Major features:

  • List view for user to navigate through all the available events
  • Clicking on item will direct user to Event detail screen
  • Searching function will look for keyword in both event title and detail fields
  • Sorting function will take the value from the spinner then send query to the database

Providing conveniet shortcut for user
