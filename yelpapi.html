Yelp API Android APP

This is Yelp API Android App used for retrieving data from yelp database through Yelp Fushion API . This App allows clients to view yelp data on the other app and store favourite store on his own preference. This APP is developed with Android Studio with Kotlin

Walk-through of the APP

Some features that were introduced in this APP:

  • Swipe left/right to navigate between store list and favourite list
  • Flash screen
  • Searching by the keywords in any fields
  • Sorting feature with ascending and descending feature
  • Favourite List to store preferred stores
Event List

Major features:

  • List view for user to navigate through all the stores
  • Clicking on item will direct user to store detail screen
  • Searching function will look for keyword in any fields
  • Sorting function will be ascending and descending if click one more time
  • Swipe left and right to navigate to favourite list screen in order to provide a user-friendly GUI
Detail Page

Providing detail information of the selected store

  • There is hyperlink on the image that user can access yelp online store resources
  • The most recenct comments is selected to put on the screen with user name, posted time and rating bars
  • Restaurant name and address are essential elments, if necessary, contact can be added as well
  • FAV! button will save the store in favourite list and toast an indication message
Favourite List

Client will see stored stores here

  • Same as store list
  • Swipe left right to navigate to store list screen
  • Click item into its detail page with a remove button inside.

Conclusion

An app built on external source and used as your own
